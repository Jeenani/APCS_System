🔧 КОНФИГУРАЦИЯ (ГЛАВНОЕ!):

Файл: apcs_system/lib/config/environment.dart

Что менять:
• apiBaseUrl = 'http://10.0.2.2:8080/api/v1'  ← API сервера
• enableServerSettings = true                   ← Settings экран
• enableApiLogging = true                       ← Логи
• isDebugMode = true                           ← Debug mode

Примеры:
• Android эмулятор: 'http://10.0.2.2:8080/api/v1'
• iOS симулятор:    'http://localhost:8080/api/v1'
• Real device:      'http://192.168.1.100:8080/api/v1'
• Production:       'https://api.example.com/api/v1'
